# 反井字遊戲 — 影片旁白逐字稿（zh-TW）

> 配音用。每段對應影片裡的一個場景，**目標秒數**是該場景動畫的長度，照著唸大致就能對上畫面。
> 全片約 **4 分 22 秒**（遠低於 12 分鐘上限）。建議語速平穩、咬字清楚；段落之間可留半秒換氣。
> 錄音請用安靜環境、麥克風靠近但避免噴麥，確保「聲音清晰」。

### 場景 1 — 題目與規則（~35s）
歡迎來到「反井字遊戲」。在一個 N×N 的棋盤上，Alice 先下、Bob 後下，兩人輪流放棋子。當 N 顆同樣顏色的棋子連成一條線，就叫做「連線」。不過這裡的規則跟一般井字遊戲相反——先連成一線的人，不是贏，而是「輸」！我們想知道：到底誰有必勝或不敗的策略？

### 場景 2 — N = 1（~14s）
先看最小的情況，N 等於 1。棋盤只有一格，Alice 一放下去就立刻連成一線，所以先手必輸。

### 場景 3 — N = 2（~19s）
N 等於 2 時，利用旋轉和對稱，前兩步其實只有兩種情形。不管是哪一種，輪到 Alice 的第三步，無論放哪一格，都會讓自己連成一線——先手一樣必輸。

### 場景 4 — N = 3、4（~17s）
到了 N 等於 3 和 4，因為局面有限，可以直接用電腦枚舉所有下法來驗證。結果是：只要雙方都採最佳策略，最後會是平手。

### 場景 5 — 結論總覽（~16s）
整理一下：N 等於 1、2 時先手必輸；而從 N 等於 3 開始，不管棋盤多大都是平手。接下來，我們就來證明這個一般情況。

### 場景 6 — 奇數：先手不會輸（~32s）
先看 N 是奇數，先證「先手不會輸」。策略是：先手第一步先佔正中心，之後對手下在哪，先手就下「中心對稱」的那一格。這樣一來，通過中心的線一定同時有雙方棋子；而其他的線，如果先手能連成，代表後手在上一步早就先連成了。所以先手永遠不會先連線。

### 場景 7 — 奇數：後手不會輸（~73s）
再來證「後手不會輸」。先觀察：棋盤上總共有 2N+2 條線，以 5×5 為例就是 12 條——五列、五行，再加上兩條對角線。關鍵想法是：只要每一條線上都同時有雙方的棋子，就沒有人能連成單色線。為了做到這件事，我們把格子兩兩「分組」，讓每一條線都至少包含某一組的兩格；圖上標同一個號碼的，就是同一組。後手的策略就很簡單：先手下哪個號碼，後手就跟著下同號碼的另一格。這樣每一組最後都是一紅一藍，每條線都混進了雙方的棋子，誰也連不成。那如果先手故意下沒有編號的格子呢？因為 N 是奇數，這種格子剛好是奇數個，後手跟著一起消耗就好。所以後手也不會輸。

### 場景 8 — 偶數（~25s）
那 N 是偶數呢？偶數棋盤沒有正中心，所以這次換成「後手」當鏡像方：先手下哪，後手就下中心對稱的位置，一樣保證後手不會輸。而「先手不會輸」，則用剛剛一模一樣的分組證明。所以偶數時也是平手。

### 場景 9 — 總結（~31s）
總結一下：N 等於 1、2 時先手必輸；只要 N 大於等於 3，無論奇偶都是平手。先手靠的是佔中心加上中心對稱，後手靠的是把格子兩兩分組配對。所以這個反井字遊戲，只要雙方都夠聰明，從三乘三開始就永遠是平手。謝謝觀看！
